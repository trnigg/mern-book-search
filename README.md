# MERN Book Search Engine
[`/mern-book-search`](https://github.com/trnigg/mern-book-search) - Module 21 Challenge

_A MERN-stack book search engine using Google Books API._

## Description

- This web-app is allows users to create an account, search for books using the Google Books API, and to save/remove books from their list.
- The app was built using the MERN stack, with a React front end, MongoDB database, and Node.js/Express.js server and API; it was refactored to use GraphQL API and Apollo Server, rather than RESTful API.
- This project uses a number of dependencies - for a full list please see [`package.json`](./package.json) in the repo.

## Installation

- No installation is necessary, simply visit [**the deployed site here.**](https://mern-book-search-24cb1c51ec50.herokuapp.com/)

## Usage

- To use the site, search for any books using the search bar at the top of the page.
- For additional features, the user can log in or sign up using the links in the top left; this will allow the user to save any searched books, and reveal a link where saved books can be viewed and/or removed.
